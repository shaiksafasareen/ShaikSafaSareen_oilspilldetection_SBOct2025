## How It Works

### Video Processing
1. Upload video file
2. Video is processed frame-by-frame
3. Original and annotated frames are stored
4. Detection information is collected for each frame

### Display After Processing
1. **Side-by-Side Videos**: Original and detected videos shown together
2. **Statistics**: Overall detection metrics
3. **Timeline Chart**: Detection pattern over time
4. **Detected Frames**: Grid of frames with detections
5. **Download Options**: Multiple export formats including PDF

### PDF Report Contents
- Video information (filename, size, frame count)
- Overall statistics
- Frame-by-frame analysis with:
  - Original frame image
  - Detected frame image (side-by-side)
  - Detection count and confidence
  - Individual detection details
- Summary section

## Usage

1. **Upload Video**: Select a video file (MP4, AVI, MOV, etc.)
2. **Process**: Wait for frame-by-frame processing
3. **Compare**: View original and detected videos side-by-side
4. **Review**: Check detected frames at the bottom
5. **Download**: Get annotated video and comprehensive PDF report
